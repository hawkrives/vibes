# How-to Guides

**Practical, goal-oriented guides for solving specific problems**

How-to guides are recipes that take you through steps to solve specific, real-world problems. Unlike tutorials, they assume you have basic knowledge and want to accomplish a particular task.

## When to Use How-to Guides

- You know what you want to achieve
- You need practical steps to solve a problem
- You want a quick solution without extensive explanation
- You're looking for best practices for a specific task

## Guide Format

Each how-to guide follows this structure:

1. **Problem statement** - What you want to achieve
2. **Prerequisites** - What you need before starting
3. **Steps** - Clear, numbered instructions
4. **Verification** - How to confirm it worked
5. **Troubleshooting** - Common issues and solutions
